作品说明：

一、项目概述
家庭应急物资清单生成器是一款基于阿里云ESA边缘生态构建的智能应急准备工具。用户通过简单的表单输入（城市、人数、居住环境、应急天数），即可获得个性化的应急物资清单，包含防灾、防疫、日常应急三大类别，每个物品都标注了数量、保质期、存储建议和采购优先级。

二、实用性
1. 解决实际问题：帮助家庭科学准备应急物资，避免盲目采购或遗漏关键物品
2. 个性化推荐：根据16个主要城市的气候和地理特征，提供针对性建议（如地震带城市推荐急救包，台风城市推荐多功能工具）
3. 智能计算：根据家庭人数和应急天数自动计算物资数量，避免浪费或不足
4. 保质期管理：自动计算有效期，提醒用户及时更新物资
5. 优先级标注：帮助用户合理安排采购顺序，优先准备高优先级物品

三、创意性
1. 3D交互场景：使用Three.js在页面右侧渲染旋转的立方体和浮动的彩色球体，配合星空背景，营造科技感
2. 粒子动画：页面背景使用TSParticles实现动态粒子效果，鼠标交互时产生吸引和排斥效果，增强参与感
3. 现代化UI：采用玻璃态设计（Glassmorphism）和渐变色彩，视觉效果美观现代
4. 响应式设计：完美适配桌面端、平板和移动端，提供一致的用户体验

四、技术探索
项目充分利用了阿里云ESA完整的边缘生态：

1. ESA Pages部署
   - 前端应用部署在ESA Pages，享受全球边缘节点加速
   - 静态资源通过CDN分发，提升访问速度

2. 边缘函数（Edge Functions）
   - 清单生成逻辑部署在边缘函数（functions/api/generate.ts）
   - 实现低延迟响应，减少服务器压力
   - 支持POST请求，返回JSON格式的清单数据

3. 边缘存储和缓存
   - 静态资源存储在边缘节点
   - 常用数据通过边缘缓存加速访问

技术栈：
- 前端：React 18 + TypeScript + Vite 5
- 3D渲染：Three.js + @react-three/fiber + @react-three/drei
- 粒子效果：@tsparticles/react + @tsparticles/slim
- 边缘函数：TypeScript + Edge Runtime
- 构建工具：Vite（支持HMR和快速构建）

技术亮点：
- 完整的TypeScript类型系统，确保代码质量
- 组件化架构，易于维护和扩展
- 边缘计算优化，提升用户体验
- 响应式设计，适配各种设备
- 3D场景和粒子效果的性能优化

五、使用说明
1. 选择居住城市（16个主要城市可选）
2. 输入家庭人数（1-20人）
3. 选择居住环境（高层/低层）
4. 设置应急天数（3-30天）
5. 点击"生成应急物资清单"按钮
6. 查看生成的清单，包含三大类别的详细物品信息
7. 可根据优先级和保质期安排采购计划
